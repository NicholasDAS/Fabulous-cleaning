# Fabulous Cleaning Service Web Application

Fabulous Cleaning is a full-stack web application designed for a professional cleaning service. 
The platform allows customers to book cleaning services online while providing an admin dashboard 
for managing bookings and customers.

## Features
User registration and login (email authentication)
Online service booking
Booking management dashboard
Email notifications (confirmation & updates)
Responsive design (mobile-friendly)
Recure authentication and session handling

# Tech Stack
**Backend:** Python (Flask)
**Frontend:** HTML, CSS, Bootstrap, JavaScript
**Database:** SQLite
**Email:** Flask-Mail / SMTP
**Authentication:** Flask-Login

## Project Structure
app.py #Main Flask application
templates/ #HTML templates
static/ #CSS, JavaScript, images
instance/ # Database files
requirements.txt #Python dependencies
